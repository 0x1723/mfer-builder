# mfer-builder

Seed

mfers are generated using a 28 length string as a "seed":

   00           00
background     type
eyes
mouth
headphones
smoke
watch
beard
hoodies
shirt
longHair
hatsUnder
hatsOver
shortHair

None: -1
Random: 00
Trait 1: 01
Trait n: n

For Example, 0000000000000000000000000000 would mean all images are random
-100000000000000000000000000 would mean there was no background
